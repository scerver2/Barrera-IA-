Barrera amb IA + Arduino 4t ESO · Projecte amb Teachable Machine

Entreneu el vostre propi model de visió per controlar una barrera amb un servo i un LED, utilitzant gestos amb el braç i Arduino.

Institut Francesc Ferrer i Guàrdia
Projecte desenvolupat per:
Salvador Cervera

🧠 1. Afegir i carregar models dels grups

A Teachable Machine ves a Export Model → TensorFlow.js → Upload (hosted) i copia la URL (acaba en /). Escriu també el nom del vostre grup.

🔌 2. Connectar amb l'Arduino

Connecta l'Arduino per USB i després dona permís al navegador.

🧪 Web Serial · Navegador recomanat: Chrome / Edge

📷 3. Iniciar IA + Webcam

Quan el model estiga carregat i l'Arduino connectat, polsa Iniciar IA + Webcam. Si les classes es diuen "Repos" i "Activar", la pàgina enviarà les lletres R i A a l'Arduino.

💡 Recorda: col·loca't davant de la càmera amb bona llum i fes gestos clars.

🎛️ 4. Vista de la càmera i detecció

A sota veuràs la webcam, les probabilitats de cada classe i la classe considerada com a "guanyadora", que és la que envia l'ordre a l'Arduino.
